# BoatHud

A Fabric mod that adds a HUD specifically for boating.

A simple in-game map mod for Minecraft, focused on navigation and ice detection in frozen biomes.

## Features
- In-game HUD map display
- Ice block detection (packed ice, blue ice)
- Configurable via Mod Menu and Cloth Config
- Supports multiple languages

This mod needs [Cloth Config](https://github.com/shedaniel/cloth-config).
To configure, you need [Mod Menu](https://github.com/TerraformersMC/ModMenu/).

# BoatHud

一个 Fabric 模组，为划船提供专用的 HUD 界面。

一个简单的 Minecraft 游戏内地图模组，专注于冻原生物群系中的导航与冰层检测。

本模组需要 [Cloth Config](https://github.com/shedaniel/cloth-config)。
如需配置，请安装 [Mod Menu](https://github.com/TerraformersMC/ModMenu/)。

## 功能
- 游戏内 HUD 地图显示
- 冰块检测（压实冰、蓝冰）
- 支持通过 Mod Menu 和 Cloth Config 配置
- 支持多语言
